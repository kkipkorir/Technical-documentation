Introduction

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML.

  • It controls the layout, colors, fonts, and other visual aspects of web pages.
  • CSS rules consist of a selector and a declaration block.
    • selector { property: value; }
Selectors and Declarations

CSS rules consist of a selector and a declaration block. The selector targets HTML elements, and the declaration block contains property-value pairs.Common types include:

  • Element Selector:
    HTMLElement { ... }
  • Class selector:
    .className { ... }
  • ID Selector:
    #idName { ... }
  • Attribute Selector:
    element[attribute="value"] { ... }
  • Descendant Selector:
    parent descendant { ... }
    • Declaration box: contains property-value pairs that define styles for the selected elements.
    h1 {
    color: blue;
    font-size: 24px; }
Box Model

Content, Padding, Border, Margin

The box model defines how elements are rendered. It includes content, padding, border, and margin.

.box {
width: 200px;
padding: 20px;
border: 2px solid black;
margin: 10px;
}
Display

The display property determines how an element is rendered.

.block { display: block;//displays elements in block format }

.inline { display: inline;// displays elements in one line }

.flex { display: flex;//makes a parent element as a container and the children as container objects }
Positioning

Elements can be positioned using position property values.There are four values

  • Static: This is the default value
    div{
    position:static;
    }
  • Absolute: Removes an element from normal document flow and postions it based on values of top,left,right and bottom eg:
    div{
    position:absolute;
    top:15px;left:180px;
    }
  • Relative: Element is positioned according to the normal flow of document but the top,left,right and bottom are active eg:
    div{
    position:relative;
    top:15px;left:180px;
    }
Typography

Font properties

Define font properties for text.

body {
font-family: 'Arial', sans-serif;
font-size: 16px;
font-weight: bold;
font-style: italic;
}

Text properties

Style text with various properties.

p {
color: #333;
text-align: center;
text-decoration: underline;
letter-spacing: 2px;
}
Colors and Background

Color Representation

Define colors using different formats

/* Hexadecimal */
.element1 { color: #ff0000; }
/* RGB */
.element2 { background-color: rgb(255, 0, 0); }
/* HSL */
.element3 { border-color: hsl(0, 100%, 50%); }

Background Properties

Style element backgrounds.

.section {
background-color: #f0f0f0;
background-image: url('background.jpg');
background-repeat: no-repeat;
background-position: center;
}
